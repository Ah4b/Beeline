Beeline QGIS Plugin

A simple plugin that makes use of the GeographicLib and lets QGIS users create lines along great circles to connect a set of input points.



  • Accepts only WGS 84
  • Ignores invalid geometries
  • Works for few hundred input points


Any issues should be logged at: https://github.com/Ah4b/Beeline/issues
